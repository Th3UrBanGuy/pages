The Challenge

SDG 8: Decent Work & Economic Growth. Many students struggle to identify relevant job opportunities, understand their skill gaps, and plan a learning journey to become employable.

The Solution

An intelligent web platform that uses AI & NLP to parse CVs, generate personalized career roadmaps, and match candidates to jobs with a transparency score.

Core Architecture

Built for the 24-Hour Onsite Challenge

Smart CV Extraction

Uses NLP to automatically parse user CVs and extract key skills, roles, and experience levels into the profile.

Intelligent Matching

Calculates a "Match %" for every job listing based on skill overlap and provides a reason (e.g., "72% Match").

AI Career Roadmap

Generates a step-by-step personalized learning path to bridge skill gaps for specific target roles.

CareerBot Assistant

An integrated chatbot mentor to answer questions like "What should I learn next for Backend Dev?"
